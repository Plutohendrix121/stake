Staking Rewards

Institutions can earn staking rewards on a broad range of proof-of-stake cryptoassets

Project
Reward %
Reward %: The total number of tokens earned if staked at the current time, expressed as the annual percentage increase of staked tokens. This is a gross value, and does not include Staked's commission. The Reward % is variable and changes with every block based on the Inflation %, Transaction Fee Rewards and the Staked % for each protocol.
Staked %
Staked %: The total number of tokens staked as a percentage of the network's total supply of tokens. Total supply is used instead of circulating supply because unvested tokens can often stake and earn rewards. The Staked % is important because only staked tokens earn staking rewards.
Inflation %
Inflation %: The number of new tokens issued, expressed as the annual percentage increase of the network's total supply of tokens. Inflation and transaction fees are the main sources for staking rewards.
